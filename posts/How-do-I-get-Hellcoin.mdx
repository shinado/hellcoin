## 1. Introduction

Hellcoin is currently only deployed on Solana via pump.fun, therefore you need to have a Solana wallet to get $HELL.    

## 2. Create a Solana wallet

You can use any Solana wallet to get $HELL. We recommend using [Phantom](https://phantom.app/) as it's the most user-friendly.

## 3. Buy $HELL

Go to [pump.fun](https://pump.fun/coin/oLMyKTuqw8foxar2b11aZf7k7f4a9M8TRme5bh8pump) and connect your wallet.
